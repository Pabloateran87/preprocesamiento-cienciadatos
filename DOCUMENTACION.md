# Documentación del Proyecto

## Introducción
Este proyecto aplica Git y GitHub para el control de versiones de un proyecto de Ciencia de Datos.

## Funcionalidades
- Eliminación de duplicados
- Manejo de valores nulos
- Codificación de variables categóricas
- Normalización de datos

## Comandos Git usados
- git clone
- git add
- git commit
- git push
- git pull
- git merge

## Automatización
Se implementó GitHub Actions para ejecutar automáticamente el script Python.
